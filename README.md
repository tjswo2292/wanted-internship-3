<h2>commit message convention</h2> 

| Commit Type | Format                                               |
| ----------- | ---------------------------------------------------- |
| feat        | 새로운 기능 추가                                     |
| bug         | 버그 수정                                            |
| design      | 사용자 UI 디자인 변경(CSS 등)                        |
| refactor    | 코드 리팩토링                                        |
| rename      | 파일 혹은 폴더명을 수정만 한 경우                    |
| remove      | 파일을 삭제만 한 경우                                |
| docs        | 문서수정                                             |
| chore       | 빌드 업무 수정, 패키지 매니저 수정(gitignore수정 등) |
